Changelog

Website version history.

Current package version: v0.0.6 Beta. This page is included so changes can be tracked visibly on the website.

v0.0.6 Beta

Release type: Advanced surrender page rebuild.

  • Rebuilt the Surrender page using the supplied advanced Shelter and Surrender page as the styling and content model.
  • Added a sticky quick navigation bar for route helper, readiness checker, animals accepted, process, FAQs and contact sections.
  • Added an interactive surrender route helper to separate owner surrender, rescue, emergency and general-help routes.
  • Added a surrender readiness checker with progress bar and print checklist option.
  • Added searchable and filterable accepted animal categories.
  • Added a visual surrender journey with outcomes and expandable FAQs.
  • Updated version references from v0.0.5 Beta to v0.0.6 Beta.

v0.0.5 Beta

Release type: Readability, mega menu and service-page expansion.

  • Fixed dark green cards where text became unreadable by restoring high-contrast white text and clearer APES green styling.
  • Checked and strengthened mega menu text contrast, card text contrast and service highlight readability across the website.
  • Renamed the Services item inside the mega menu to Service Finder while keeping the top navigation button as Services.
  • Added under-construction Trial Adoptions and Foster Scheme pages.
  • Added Trial Adoptions and Foster Scheme links to the Services mega menu, Services directory and Adoptions page.
  • Updated version references from v0.0.4 Beta to v0.0.5 Beta.

v0.0.4 Beta

Release type: Adoption page and fee calculator enhancement.

  • Enhanced the Adoptions page with a more engaging hero, adoption journey timeline, highlight cards and adopter guidance sections.
  • Expanded the adoption fee explorer to include all listed animal fee types: snakes, lizards, rats, mice, rabbits, guinea pigs, standard birds, exotic birds, spiders, scorpions and tailless whip scorpions.
  • Rebuilt the adoption fee calculator with all animal fee options, the fixed £15 admin fee, optional add-ons and optional £30 reservation fee planning.
  • Updated the estimate total box with APES green and gold styling, live summary chips and a clearer cost breakdown.
  • Added generic filter behaviour for searchable fee cards and other filterable site sections.
  • Updated version references from v0.0.3 Beta to v0.0.4 Beta.

v0.0.3 Beta

Release type: Mega menu interaction fix.

  • Changed the Services mega menu to click-open behaviour so it stays open once selected.
  • Removed hover-only closing behaviour that caused the menu to disappear while moving the mouse from the button to the panel.
  • Added outside-click closing so the menu only closes when the user clicks outside the Services menu area.
  • Added Escape key support for keyboard users.
  • Updated version references from v0.0.2 Beta to v0.0.3 Beta.

v0.0.2 Beta

Release type: Layout, readability and support-widget update.

  • Fixed the homepage hero spacing issue that made the opening banner appear too squashed.
  • Updated the site wrapper and page grids so the website uses the full available screen width more effectively across all pages.
  • Adjusted hero sections, cards, route-helper panels, service grids and footer columns to compensate for the wider layout.
  • Improved readability by strengthening spacing, line lengths, card padding and responsive breakpoints while retaining APES green and gold brand colours.
  • Added the Chatwoot live support widget script across all public HTML pages.
  • Updated version references from v0.0.1 Beta to v0.0.2 Beta.

v0.0.1 Beta

Release type: Route-helper and readability update.

  • Created green APES Shelter & Rescue theme using supplied APES Shelter logo.
  • Added responsive mega menu across all pages.
  • Created homepage, services, adoptions, surrender, rescues, sponsorships, education, news, contact, policies, donate, portal, search and changelog pages.
  • Preserved external ShelterManager routes for adoptable animals, surrender, lost pet and found pet forms.
  • Added adoption fee search/filter and simple fee calculator.
  • Added .htaccess redirect helper for common legacy URLs.
  • Added upload-only README for simple handover.
  • Renamed the mega menu button from Service Finder to Services and removed public deployment wording from website pages.

Next release ideas

  • Replace external policy links with full static policy pages once final policy text is approved.
  • Add optimised animal and campaign images.
  • Confirm donation provider and newsletter embed details.
  • Run WCAG 2.2 AA accessibility review before public launch.
